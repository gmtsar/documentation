align_SAT.csh     
*************      

Synopsis
--------
**align_SAT.csh**  *data_type(RAW/SLC) align.in*             


Description
-----------
**align_SAT.csh** aligns a secondary image to the specified master/reference image

Required Arguments
------------------

*data_type*

	Input data type (e.g., RAW, SLC)

*align.in*

	Input file list

Example
-------
 ::

    **align_SAT.csh**
