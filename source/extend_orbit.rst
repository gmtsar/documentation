Synopsis
--------
**extend_orbit** *file_in.LED file_out.LED frac*


Description
-----------
**extend_orbit**                      

Required Arguments
------------------
    
*orig.LED*     

	Original LED file 

*extend.LED*   

	Extended LED file 

*frac*         

	Fraction of frame length to add to each end (e.g. 0.25) 


Example
-------
 ::

   extend_orbit
