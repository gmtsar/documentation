.. index:: ! split_aperture       

**************    
split_aperture    
**************    

Synopsis
--------
**split_aperture** *prm_file*  


Description
-----------
**split_aperture** Program used to split azimuth spectrum        
   
SLCs are bandpassed output is SLCF (forward) SLCB (backward) 

Required Argument
-----------------

*prm_file*

	Input parameter (PRM) file

Example
-------
 ::

    split_aperture
